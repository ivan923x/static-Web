    * <code>4</code> - <code>已結案</code>

    * <code>0</code> - <code>fa-clipboard-plus</code>
    * <code>1</code> - <code>fa-clipboard-user</code>
    * <code>2</code> - <code>fa-clipboard-clock</code>
    * <code>3</code> - <code>fa-clipboard-search</code>
    * <code>4</code> - <code>fa-clipboard-check-alt</code>
    *
    * @example
    * APP_QUERY_PKG.query_request_status_badge_icon(#STATUS#); 
    *
    * @param p_status <em>Number</em> 狀態號碼
    * @returns <em>Varchar</em> 圖示名稱字串

        ) Return VARCHAR2;
